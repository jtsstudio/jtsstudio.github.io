How to make an desktop application with electron and using an HTML file

Instructions:

  1. Download node.js here.
  2. Run command prompt, navigate to your project folder using the "cd" command.
  3. Type command: npm init -y
  4. Type command: npm install electron -D
  5. Change your package.json file: under "scripts", add "start": "electron ."
